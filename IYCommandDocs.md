## rejoin
Command Arguments: N/A

Aliases: `rj`

Rejoins the game you are playing on the server you were playing on.

## exit
Command Arguments: N/A

Aliases: N/A

Kills the Roblox process. 

## logs
Command Arguments: N/A

Aliases: `chatlogs`

Opens a window that will log what players say or whisper.

## serverinfo
Command Arguments: N/A

Aliases: `info`, `sinfo`

Opens a window that gives you the following information: 
- LocalPlayer User ID
- LocalPlayer Appearance ID
- Server Run Time
- Number of players in the game
- Maximum players allowed in the game
- Place ID
- Place Name

## explorer
Command Arguments: N/A

Aliases: `dex`

Opens DEX (v3 light theme) explorer

## remotespy
Command Arguments: N/A

Aliases: `remotes`

Starts a remotespy that outputs results to the Roblox console. (Press `F9` to open the Roblox Console).

## breakloops

Command Arugments: N/A

Aliases: `break`

Terminates all command loops made with FE Infinite Yield

## addalias
Command Arguments: `addalias [cmd] [alias]`

Aliases: N/A

Adds [alias] as an alias for the command [cmd].

## removealias
Command Arguments: `removealias [alias]`

Aliases: N/A

Removes the custom alias [alias].

## clraliases
Command Arguments: N/A

Aliases: N/A

Removes all custom aliases.

## noclip
Command Arguments: N/A

Aliases: N/A

Enables noclip. Noclip allows you to pass through solid objects.

## clip
Command Arguments: N/A

Aliases: `unnoclip`

Disables noclip. Prevents you from passing through solid objects.

## fly
Command Arguments: N/A

Aliases: N/A

Makes you fly. When flying you move with your normal controls.

## unfly
Command Arguments: N/A

Aliases: `nofly`

Disables flying.

## flyspeed
Command Arguments: `flyspeed [num]`

Aliases: `flysp`

Sets your flying speed to [num].

## float
Command Arguments: N/A

Aliases: N/A

Makes you float. When floating, you can jump to float higher.

## unfloat
Command Arguments: N/A

Aliases: `nofloat`

Disables floating.

## savepos
Command Arguments: `savepos [name]`

Aliases: `spos`, `saveposition`, `setwaypoint`

Sets a waypoint at your position with the specified name. The name is used to load the waypoint later.

## loadpos
Command Arguments: `loadpos [name]`

Aliases: `lpos`, `loadposition`, `waypoint`

Teleports you to a waypoint with the specified name you created earlier.

## deletepos
Command Arguments: `deletepos [name]`

Aliases: `dpos`, `deleteposition`, `deletewaypoint`

Deletes a waypoint with the specified name.

## clearpos
Command Arguments: N/A

Aliases: `clrpositions`, `clearpositions`, `cpos`, `clrpos`

Clears all of the waypoints you have created.

## spam
Command Arguments: `spam [text]`

Aliases: N/A

Spams the chat with the specified [text]

## nospam
Command Arguments: N/A

Aliases: `unspam`

Disables the chat spammer.

## pmspam
Command Arguments: `pmspam [plr] [text]`

Aliases: N/A

Spam whispers [plr] with [text]

## nopmspam
Command Arguments: N/A

Aliases: `unpmspam`

Turns off the PM spammer.

## esp
Command Arguments: `esp [plr]`

* If you don't specify the [plr] argument, the default is all players.

Aliases: N/A

ESP stands for Enemy Sound Position. It shows you where the player is (through walls) and stats about the player. This command enables the ESP for the specified player.

## noesp
Command Arguments: `noesp [plr]`
